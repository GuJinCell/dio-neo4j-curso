# Projetos do curso de Neo4j (DIO)

Este repositório reúne os projetos e modelos desenvolvidos durante o curso de **Neo4j** na plataforma **DIO**.

## Conteúdo
- `Relação de filmes/`: grafo de filmes, atores e diretores (arquivos do Arrows e imagem).
- `Recomendação de música/`: modelo de recomendação de música (JSON de dados e JSON para importar no Arrows).
